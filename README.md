**Playcode** is an interactive educational platform designed to introduce children to programming concepts through gamification. Built during the hackathon **VaultHiest** hosted by CGC, it reached the **3rd evaluation** for its innovative approach to logic building.

## üéÆ How It Works

Instead of writing syntax, kids use **Drag-and-Drop Blocks** to control a robot character. The goal is to solve puzzles (maze navigation) using logic loops and commands.

* **Visual Coding:** Powered by Google's **Blockly** engine.
* **Logic Challenges:** Progressive difficulty levels (Move $\rightarrow$ Loop $\rightarrow$ Conditionals).
* **Immediate Feedback:** Real-time execution of block logic on the canvas.

## ‚ú® Key Features

1.  **Custom Block Definitions:** Created simplified blocks (`Move Forward`, `Turn`, `Repeat`) tailored for children, stripping away complex syntax.
2.  **Live Code Generation:** Translates visual blocks into executable JavaScript logic in real-time.
3.  **Global Leaderboard:** Integrated **Supabase** to track completion times and high scores across users.
4.  **Interactive Game Loop:** Custom game engine that interprets block commands to animate the character sprite on a grid.
